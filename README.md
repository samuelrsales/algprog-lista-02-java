# Lista de Exercícios II - Programação de Soluções Computacionais

Este repositório contém as resoluções da segunda lista de exercícios da disciplina de **Algoritmos e Programação**, do curso de TI no **Centro Universitário UNA**.

## 📋 Informações do Projeto
- **Professor:** Daniel Henrique Matos de Paiva
- **Linguagem utilizada:** Java
- **IDE:** Visual Studio Code

## 📝 Regras de Entrega
- Realizada em equipes de até 05 alunos.
- Algoritmos devidamente indentados.
- Entrega individual conforme solicitado.

## 🚀 Exercícios Resolvidos
1. **Mensagem de Número:** Solicita um número e o exibe no console.
2. **Operações Básicas:** Pede dois números e mostra a soma, subtração e multiplicação.
3. **Média Bimestral:** Calcula a média a partir de 4 notas informadas.
4. **Cálculo de Combustível:** Calcula o valor total pago com base no preço do litro e quantidade vendida.
5. **Eurotrip:** Calcula o custo total de uma viagem para três destinos (Alemanha, Portugal e Itália) considerando o preço por destino e a quantidade de pessoas.
